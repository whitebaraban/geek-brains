discard - отмена изменений
revert - отмена коммита
reset - откат к выбранному коммиту

Создание новой версии и переключение между ветками:
Local Branches -> add Branch
double click -> checkout branch

Объединение веток:
checkout branch, в которую нужно подтянуть инфу.
merge - по ветке, которую нужно подтянуть в выбранную ветку

Копирование в мастер одного коммита:
checkout master branch
right mouse button - Cherry-Pick
